Thaliedje package¶

Spotipy¶

The Thaliedje package uses spotipy for the connection to Spotify. Note that the spotipy package uses caching files for caching the Spotify OAuth2 tokens. When using the default settings, a cache folder will be created in the website folder storing these caching files.
